ОПЦИИ
		
-x [<unsigned int>]
	Задать размер решётки по х. 			(По умолчанию: 10)

-y [<unsigned int>]
	Задать размер решётки по х.			(По умолчанию: 10)

-seed [<int>]
	Задать зерно генератора случайных чисел.	(По умолчанию: 2107)

-iter [<unsigned int>]
	Задать количество итераций			(По умолчанию: 0)
	
	Если не указывать, симуляция продлится до тех 
	пор, пока не будет достигнуто стационарное 
	состояние (осцилляторы не считаются за стац-
	ионарное состояние).
	
	Если указать, симуляция продлится до указанной 
	итерации, или пока не будет достигнуто стационарное 
	состояние.

-tests [<unsigned int>]
	Задать количество экспериментов			(По умолчанию: 1)
	
	Задаёт количество повторений симуляции
	Удобно использовать для получения более
	объективных данных.

-mode [none|firstlast|all]
	Полнота вывода.					(По умолчанию: all)
	
	none: 		ничего не выводить, кроме результатов замеров.
	firstlast:	выводить первый и последний кадр симуляции.
	all:		выводить всё.

-deadchar [<char>]
	Задать символ мёртвой клетки 			(По умолчанию: '.')

-alivechar [<char>]
	Задать символ живой клетки 			(По умолчанию: 'o')

-borderchar [<char>]
	Задать символ пограничной клетки 		(По умолчанию: 'X')
